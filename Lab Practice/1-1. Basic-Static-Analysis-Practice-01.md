# Analysis

- The file is a 32-bit PE file, which can be detected by most AV engines and be marked as bulz or cobra trojan.
- The malware is an unpacked sample.
- The malware has capability to establish HTTP connection and download file into disk through HTTP from the internet. Also, it has ability to delete itself and delay the payload execution to evade detection. The interesting part is that the malware uses `ping` command to replace `sleep()` to achieve the same objective.
- The payload is highly likely to be `C:\Users\Public\Documents\CR433101.dat.exe`
- According to the PBD path, , identified, we can assume
	- The developer name is Matt
	- The project name is `HuskyHacks / PMAT-maldev`
	- The name of file released the purpose of the malware is to download payloads from URL, and it might indicate the malware is a red team testing tool or a malware sample for education purpose as we didn't see any obfuscation techniques in the file naming as well as the malware code and strings.
