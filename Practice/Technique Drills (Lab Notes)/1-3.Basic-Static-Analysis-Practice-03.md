# Objective

Conduct basic static analysis against the malware sample `Malware.Packed.exe.malz` provided by PMAT course and record the key finds.

Tools for the lab:
- VirusTotal
- FLOSS
- Detect It Easy (DIE)
- PEView
- PEStudio
- CAPA

Malware Sample Source: [PMAT-labs/labs/1-1.BasicStaticAnalysis/Malware.PackedAndNotPacked.exe.malz](https://github.com/HuskyHacks/PMAT-labs/tree/main/labs/1-1.BasicStaticAnalysis/Malware.PackedAndNotPacked.exe.malz)
